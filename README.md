# 🧴 피부 성분 분석기

## 📋 프로젝트 개요
- **이름**: 피부 성분 분석기 (Skin Analyzer)
- **목표**: 화장품 성분표를 촬영하여 AI가 성분을 분석하고 안전성을 평가하는 웹 애플리케이션
- **주요 기능**: 카메라 촬영, 이미지 업로드, OCR 기반 성분 인식, 성분 안전성 분석

## 🌐 서비스 URL
- **개발 서버**: https://3000-igspb00beqqqykyv9tso2-6532622b.e2b.dev
- **API 엔드포인트**:
  - `POST /api/analyze` - 이미지 분석 및 맞춤 추천
  - `GET /api/ingredients` - 전체 성분 데이터베이스 조회
  - `GET /api/ingredient/{name}` - 특정 성분 정보 조회
  - `GET /api/skin-types/{type}` - 피부 타입별 추천 정보
  - `GET /api/products/recommend` - 제품 추천 (피부타입, 카테고리, 가격 필터)
  - `GET /api/skincare-clinics` - 근처 피부관리실 검색
  - `GET /api/homecare-sites` - 홈케어 쇼핑몰 목록

## ✨ 현재 구현된 기능

### 🔍 **성분 분석 기능**
- ✅ **카메라 촬영**: 전면/후면 카메라 선택 및 실시간 전환 가능
- ✅ **이미지 업로드**: 갤러리에서 이미지 파일 선택
- ✅ **OCR 시뮬레이션**: 이미지에서 성분명 추출 (현재 시뮬레이션)
- ✅ **성분 안전성 분석**: 12가지 주요 화장품 성분 데이터베이스 기반 분석
- ✅ **결과 시각화**: 안전성별 성분 분류 및 상세 설명

### 🎯 **맞춤형 추천 서비스**
- ✅ **피부 타입 분석**: 성분 기반 자동 피부 타입 추천
- ✅ **맞춤 가이드**: 6가지 피부 타입별 성분 추천 및 주의사항
- ✅ **제품 추천**: 분석된 성분과 피부 타입에 맞는 유사 제품 추천
- ✅ **가격 비교**: 추천 제품의 가격 및 평점 정보 제공

### 🗺️ **지역 서비스 연동**
- ✅ **피부관리실 찾기**: 위치 기반 근처 피부관리실 검색
- ✅ **전문가 정보**: 피부관리실별 전문분야 및 연락처 제공
- ✅ **홈케어 쇼핑몰**: 4개 주요 화장품 쇼핑몰 바로가기

### 📚 **사용자 편의 기능**
- ✅ **분석 기록 저장**: 최대 10개 분석 결과 로컬 저장
- ✅ **기록 관리**: 저장된 분석 재열람, 재분석, 삭제 기능
- ✅ **스마트 카메라**: 전면/후면 자동 감지 및 선택, 좌우 반전 처리
- ✅ **모바일 최적화**: 풀스크린 카메라, 하단 네비게이션, 터치 제스처
- ✅ **PWA 지원**: 홈화면 설치, 오프라인 모드, 햅틱 피드백

## 📊 데이터 아키텍처

### 🧪 **성분 데이터베이스**
- **성분 정보**: 12가지 주요 화장품 성분
- **속성**: 안전성(safe/caution/harmful/unknown), 효능, 적합 피부타입, 주의사항
- **카테고리**: 보습, 미백, 안티에이징, 트러블케어, 각질케어, 항산화, 방부제

- **타입**: 지성, 건성, 복합성, 민감성, 트러블, 노화피부 (6가지)
- **추천 로직**: 성분 기반 자동 피부타입 매칭 알고리즘
- **맞춤 가이드**: 타입별 추천 성분 및 피해야 할 성분 정보

### 🛍️ **제품 데이터**
- **제품 정보**: 4개 샘플 제품 (실제 서비스에서는 API 연동)
- **속성**: 브랜드, 카테고리, 가격, 평점, 리뷰수, 적합 피부타입
- **필터링**: 피부타입, 카테고리, 가격대별 검색 지원

### 🏪 **서비스 연동**
- **쇼핑몰**: 올리브영, 화해, 버드박스, 글로우픽 (4개 사이트)
- **피부관리실**: Mock 데이터 (실제로는 Google Places API 연동 예정)
- **사용자 데이터**: 로컬 스토리지 기반 (최대 10개 분석 기록)

## 🎯 사용법

### 📱 **모바일 앱 사용법**
1. **홈 화면**: 그라디언트 버튼으로 카메라 시작 또는 갤러리 선택
2. **풀스크린 카메라**: 전면/후면 전환하며 성분표 촬영
3. **제스처 조작**: 
   - 📱 더블 탭으로 카메라 전환
   - ⬆️ 위로 스와이프하여 상세 결과 보기
   - ⬇️ 아래로 스와이프하여 카메라 닫기
4. **하단 네비게이션**: 홈/촬영/기록/정보 탭으로 빠른 이동

### 📷 **기본 분석 과정**
1. **성분표 준비**: 화장품 뒷면의 성분표가 선명하게 보이도록 준비
2. **이미지 입력**: 카메라로 촬영하거나 갤러리에서 이미지 선택  
3. **분석 시작**: '성분 분석 시작' 버튼 클릭
4. **종합 결과 확인**: 
   - ✅ 성분별 안전성 분석
   - 🎯 피부 타입 맞춤 추천  
   - 🛍️ 유사 제품 추천
   - 🏠 홈케어 쇼핑몰 바로가기

### 🗺️ **추가 서비스 이용**
5. **피부관리실 찾기**: 위치 권한 허용 후 근처 전문 관리실 검색
6. **분석 기록 저장**: 결과 저장하여 나중에 재확인 및 비교 분석
7. **기록 관리**: '분석 기록 보기'에서 과거 분석 결과 관리
