## 💡 구체적 실행 가이드

### Step 1: 기존 계정 이메일 찾기

**방법 1: 이메일 검색**
```
Gmail/Naver 등에서 검색:
"Cloudflare" 또는 "carejoa" 또는 "Pages"

찾을 메일:
- "Welcome to Cloudflare"
- "Your project carejoa-kr is ready"
- "Domain carejoa.kr has been added"
```

**방법 2: 도메인 등록 정보 확인**
```
carejoa.kr 도메인을 어디서 구입했나요?
→ 그 이메일이 Cloudflare 계정일 가능성 높음
```

**방법 3: 브라우저 자동 로그인**
```
https://dash.cloudflare.com 접속
→ 자동 로그인되는 계정 확인
```

---

### Step 2: API 토큰 생성 (계정 A)

#### 2-1. Cloudflare 로그인
```
https://dash.cloudflare.com
→ 계정 A 이메일로 로그인
```

#### 2-2. API 토큰 생성
```
1. 우측 상단 프로필 아이콘 클릭
2. "API Tokens" 선택
3. "Create Token" 클릭
4. "Edit Cloudflare Workers" 템플릿 선택
```

#### 2-3. 권한 설정
```
필수 권한:
✅ Account - Cloudflare Pages - Edit
✅ Account - D1 - Edit
✅ Account - Workers Scripts - Edit
✅ Account - Account Settings - Read

선택 권한 (있으면 좋음):
✅ User - User Details - Read
✅ Account - Workers R2 Storage - Edit (향후 파일 업로드용)
```

#### 2-4. Account Resources
```
Include: All accounts
또는
Include: Specific account → 계정 A 선택
```

#### 2-5. 토큰 생성 및 복사
```
"Continue to summary" → "Create Token"
→ 토큰 복사 (한 번만 표시됨!)
```

---

### Step 3: GenSpark Deploy 탭 업데이트

```
1. GenSpark AI 접속
2. Deploy 탭 클릭
3. "Cloudflare API Token" 입력란 찾기
4. 새 토큰 붙여넣기
5. 저장
```

---

### Step 4: 배포 테스트

```bash
cd /home/user/webapp
npm run build
npx wrangler pages deploy dist --project-name carejoa-kr
```

**예상 결과:**
```
✨ Deployment complete!
🌍 https://carejoa.kr
```
