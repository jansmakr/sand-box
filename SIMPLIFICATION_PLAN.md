# 케어조아 단순화 계획

## 핵심 원칙
- 큰 글씨 (최소 20px)
- 큰 버튼 (최소 높이 80px)
- 최대 3가지 주요 기능만
- 아이콘 + 명확한 한글 텍스트

## 메인 페이지 개선

### 헤더
- 로고 + 로그인/내정보 버튼만

### 메인 콘텐츠 (3개 큰 버튼)
1. **요양시설 찾기** (파란색, 가장 큼)
   - "우리 부모님께 맞는 요양시설을 찾아드립니다"
   
2. **전화 상담** (초록색)
   - "1577-XXXX"
   - "전문 상담원이 도와드립니다"

3. **카카오톡 상담** (노란색)
   - "편하게 문의하세요"

### 제거할 것들
- 복잡한 모달/팝업
- 작은 글씨의 부가 설명
- 여러 개의 작은 버튼들
- 통계/수치 표시

## 마이페이지 개선

### 3가지 메뉴만
1. 내 정보 수정
2. 견적 신청
3. 로그아웃

## 로그인 페이지
- 카카오 로그인 버튼 하나만 (크게)

## 견적 신청 폼
- 필수 항목만 5개 이하
  1. 환자 이름
  2. 나이
  3. 지역 (시/도만)
  4. 시설 종류 (요양원/재가복지)
  5. 연락처
