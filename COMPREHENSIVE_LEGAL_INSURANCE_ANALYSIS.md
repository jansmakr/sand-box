## 2. 서비스별 법적 리스크 평가

### 🔴 치명적 리스크 (P0 - 즉시 조치 필수)

#### **2.1 응급 이송 서비스 (Risk Score: 95/100)**

**법적 문제:**
```
⚠️ 응급의료법 위반 가능성 매우 높음

1. 응급의료법 제48조 (응급환자 이송)
   - 응급환자 이송은 "구급차등"만 가능
   - 케어조아가 이송을 "중개"하는 행위도 위법 가능성
   
2. 의료법 제27조 (무면허 의료행위)
   - 응급 상황 판단은 의료행위에 해당
   - 플랫폼이 "응급"을 판단하면 무면허 의료행위

3. 형사 책임
   - 응급 이송 중 사망/후유증 → 과실치사상 가능
   - 최대 징역 5년 또는 벌금 5천만원
```

**권장 조치:**
```
✅ 즉시 서비스 중단 (법률 자문 완료 전까지)
✅ "응급" 키워드 제거 → "일반 이송"으로 변경
✅ 의료기관 승인 필수 조건 추가
✅ 면책 조항 강화 (판단 책임은 의료기관)
```

**대체 방안:**
```
Option 1: 일반 이송 서비스로 축소
  - "응급" 제거 → "병원 간 일반 이송"
  - 의료진 동승 필수 명시
  - 119 연계 권장

Option 2: 완전 폐지
  - 가장 안전한 선택
  - 법적 리스크 제거
```
