## 🎯 3가지 데이터 수집 방법

### 방법 1: 파일 다운로드 (권장 - 즉시 사용 가능) ⭐

#### Step 1: CSV 파일 다운로드
1. 공공데이터포털 접속
2. 검색: **"국민건강보험공단_장기요양기관"**
3. **파일데이터** 탭 클릭
4. **"국민건강보험공단_장기요양기관 시설별 현황"** 다운로드
5. 파일에 포함된 정보:
   - 기관명, 주소, 시설구분
   - 정원, 현원 (공실 계산 가능)
   - 평가등급 (일부 파일)
   - 시도, 시군구

#### Step 2: 케어조아에 업로드
```bash
# 1. 다운로드한 CSV 파일을 public-data.csv로 저장
cp ~/Downloads/장기요양기관.csv /home/user/webapp/public-data.csv

# 2. CSV → SQL 변환 스크립트 실행
cd /home/user/webapp
node scripts/import-public-data.cjs

# 3. 생성된 SQL 파일로 DB에 로드
npx wrangler d1 execute carejoa-production --local --file=./import-public-data.sql
```

#### 장점
- ✅ API 키 승인 없이 즉시 사용 가능
- ✅ 전국 모든 시설 데이터 (15,000개 이상)
- ✅ 평가등급 포함
- ✅ 무료
